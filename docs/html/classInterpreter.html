◆ hasExecutionHalted()

bool Interpreter::hasExecutionHalted ( )

Checks if the Interpreter is currently halted

Due to the WAIT instruction (0xFX0A), the Interpreter may halt until a key is pressed. This function determines if the Interpreter is currently halted, waiting for input.
